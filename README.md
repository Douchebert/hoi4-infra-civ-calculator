# HOI4 Infra + Civ Factory Build Order Calculator

A tool to help plan optimal construction of **infrastructure** and **civilian factories** in Hearts of Iron IV.

## Goal
Make an easy-to-use calculator where you input your current situation and it suggests a good build order / priorities.

## Features (to be added)
- Input current number of civilian factories
- Input infrastructure levels and available building slots
- Target goals (e.g. reach X civs by certain date)
- Calculate construction times and output

Learning project — feel free to contribute!
